# 📊 Super Store Sales Dashboard

## 📌 Overview
Interactive dashboard for analyzing sales, profit, and customer trends across regions, categories, and segments.

## 🎯 Key Metrics
- **Sales:** $1.6M  
- **Orders:** 22K  
- **Profit:** $175K  
- **Avg. Ship Days:** 4  

## 📊 Key Insights
- COD is the most used payment method (43%)  
- West region leads in sales  
- Consumer segment contributes the most revenue  
- Sales and profit peak in Q4  
- Office Supplies is the top category  

## 🛠️ Tools Used
- Power BI / Tableau / Excel  
- Data Visualization  

## 🚀 Usage
Open the dashboard file and use filters to explore trends by region, segment, and time.
